# Alternate Universes - Lightspeed Custom CSS

Custom CSS styling for the Alternate Universes Lightspeed store.

## Files

- `lightspeed-custom.css` - Brand-specific styles using CSS variables for easier maintenance

## Usage

Load `lightspeed-custom.css` in your Lightspeed store settings after the base Lightspeed CSS.

## Customization

All colors and typography are defined as CSS variables in the `:root` section. Update these variables to change the store's appearance globally.
